# usb-rotary-keyboard

Keyboard Mappings:


| Rotary | Left           | Right      | Click      |
|--------|----------------|------------|------------|
| 1      | Volume Down    | Volume Up  | Mute       |
| 2      | Media Previous | Media Next | Pause/Play |
| 3      | F14            | F13        | F15        |
| 4      | F17            | F16        | F18        |
