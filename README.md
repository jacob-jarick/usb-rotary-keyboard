# usb-rotary-keyboard

Keyboard Mappings:


| Rotary | Left           | Right           | Click      |
|--------|----------------|-----------------|------------|
| 1      | Volume Down    | Volume Up       | Mute       |
| 2      | Media Previous | Media Next      | Pause/Play |
| 3      | Media FFWD     | Media RWD       | Media Stop |
| 4      | KEY_LEFT_ARROW | KEY_RIGHT_ARROW | SPACE      |
